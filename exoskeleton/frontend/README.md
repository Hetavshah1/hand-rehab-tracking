# Exoskeleton Therapy Progress Tracker Frontend

## Setup Instructions

1. **Install dependencies** (if not already done):
   ```sh
   npm install
   ```

2. **Start the React app:**
   ```sh
   npm start
   ```
   The app will run at [http://localhost:3000](http://localhost:3000)

3. **Connects to backend:**
   - Make sure your Flask backend is running at `http://localhost:5000`
   - The frontend will communicate with the backend for authentication, patient/doctor portals, and progress tracking.

## Features
- Login & Signup (doctor/patient)
- Doctor dashboard: view all patients, patient profiles, add sessions/progress, view graphs
- Patient dashboard: view own profile, sessions, progress
- Profile page for both roles
- Responsive, modern UI

---

**Start both backend and frontend for full functionality!**
